# 앱 테스트 링크

## 웹 버전 테스트 (개발/테스트용)

### 모바일 앱 웹 버전
- **URL:** http://43.200.44.109:3001
- **용도:** 웹 브라우저에서 모바일 앱 테스트
- **접근 방법:** 브라우저에서 위 URL 접속

- **예시:** http://43.200.44.109:3001/invite.html?invite=cmkatfk0r0001sqqsrlnk7oek

- **예시:** http://43.200.44.109:3001/contract/cmkatfk0r0001sqqsrlnk7oek
- **용도:** 계약서 서명 테스트

## APK 다운로드 (네이티브 앱)

### 현재 APK 파일 위치
- **로컬 경로:** `apps/mobile/android/app/build/outputs/apk/release/app-release.apk`
- **파일 크기:** 약 7.5MB

### APK 설치 방법

#### 방법 1: 직접 전송
1. APK 파일을 핸드폰으로 전송 (이메일, 카카오톡, AirDrop 등)
2. 파일 관리자에서 APK 파일 클릭
3. "알 수 없는 출처" 허용
4. 설치 완료

#### 방법 2: 서버에 업로드 후 다운로드 링크 제공
APK 파일을 서버에 업로드하면 다운로드 링크를 제공할 수 있습니다.

## 테스트 시나리오

### 시나리오 1: 핸드폰 인증 로그인
1. 앱 실행
2. 핸드폰 번호 입력: `010-1234-1234`
3. 인증번호 입력: `1234`
4. 로그인 성공 → 메인 화면 표시

### 시나리오 2: 카카오톡 계약서 링크
1. 카카오톡에서 계약서 링크 클릭
2. 앱으로 연결 (deep link)
3. 계약서 서명
4. 자동 로그인 → 메인 화면 표시

### 시나리오 3: 초대 링크로 등록
1. 초대 링크 클릭
2. 근로자 정보 입력
3. 등록 완료 → 메인 화면 표시
