# 최종 답변

## 현재 문제

**Admin 웹의 프론트엔드 코드**가 포트 3000을 사용하고 있습니다.
- 문제: 프론트엔드 코드 (packages/shared/api.ts)
- 해결: 프론트엔드 코드 수정 및 재배포

**백엔드는 이미 포트 3002에서 실행 중입니다.**

## 확인 순서

### 1. 백엔드 상태 확인 (먼저)

```bash
./check-backend.sh
```

이 스크립트는:
- PM2 프로세스 상태 확인
- 백엔드 헬스 체크 (포트 3002)
- 포트 사용 현황 확인
- .env 파일 확인

### 2. 백엔드가 정상이면

Admin 웹만 재배포:

```bash
./redeploy-admin.sh
```

### 3. 백엔드가 비정상이면

백엔드도 재배포해야 합니다.

## 결론

**현재 문제는 프론트엔드 코드이므로, Admin 웹만 재배포하면 됩니다.**

하지만 백엔드 상태도 확인하는 것이 좋습니다.
