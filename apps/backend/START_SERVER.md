# 백엔드 서버 시작 가이드

## 포트 충돌 해결

포트 3000이 이미 사용 중인 경우:

```bash
# 포트 3000을 사용하는 프로세스 종료
lsof -ti:3000 | xargs kill -9

# 또는 특정 프로세스 ID를 종료
kill -9 [프로세스ID]
```

## 서버 시작

```bash
cd "/Users/yoojihyeon/Desktop/사업들/ECO spot/장애인 근로관리 프로그램/어드민(장애인근로관리))/apps/backend"
npm run start:dev
```

터미널에 에러 메시지가 출력됩니다.
