# 백엔드 서버 에러 확인 방법

## 방법 1: 백엔드 디렉토리에서 서버를 포그라운드로 실행 (가장 추천)

```bash
cd apps/backend
npm run start:dev
```

이렇게 하면 터미널에 에러 메시지가 직접 출력됩니다.
에러가 발생하면 Ctrl+C로 서버를 중지할 수 있습니다.

## 방법 2: 실행 중인 프로세스의 로그 확인 (백그라운드로 실행한 경우)

먼저 실행 중인 프로세스를 확인:
```bash
lsof -ti:3000
```

프로세스가 실행 중이면, 해당 프로세스를 종료하고 다시 포그라운드로 실행하세요:
```bash
lsof -ti:3000 | xargs kill -9
cd apps/backend
npm run start:dev
```

## 방법 3: 실행 중인 백엔드 서버가 있는 터미널 찾기

백엔드 서버를 실행한 터미널 창을 찾아서 직접 확인하세요.
그 터미널에 에러 메시지가 표시될 것입니다.
