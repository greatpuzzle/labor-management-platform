# 키스토어 생성 시 입력 정보 가이드

## keytool 질문 항목 설명

키스토어 생성 시 다음과 같은 정보를 입력하게 됩니다:

### 1. 이름과 성 (CN - Common Name)
**권장:** 회사 이름 또는 앱 이름
- 예: `ECO Spot` 또는 `Labor Management`
- 개인 이름을 넣어도 되지만, 보통은 회사/조직 이름 사용

### 2. 조직 단위 (OU - Organizational Unit)
**권장:** 부서 또는 팀 이름
- 예: `개발팀`, `IT팀`, `Development`

### 3. 조직 (O - Organization)
**권장:** 회사 이름
- 예: `ECO Spot`, `(주)에코스팟`

### 4. 도시 또는 지역 (L - Locality)
**권장:** 회사 위치 도시
- 예: `서울`, `부산`

### 5. 주 또는 도 (ST - State)
**권장:** 주 또는 도 이름
- 예: `서울`, `경기도`

### 6. 국가 코드 (C - Country)
**필수:** 2자리 국가 코드
- 한국: `KR`
- 미국: `US`
- 일본: `JP`

## 예시 입력

```
이름과 성: ECO Spot
조직 단위: 개발팀
조직: ECO Spot
도시: 서울
주/도: 서울
국가 코드: KR
```

또는

```
이름과 성: Labor Management
조직 단위: Development
조직: ECO Spot
도시: Seoul
주/도: Seoul
국가 코드: KR
```

## 개인 개발자인 경우

개인 개발자라면:
- 이름과 성: 개인 이름 또는 앱 이름
- 조직: 개인 이름 또는 "개인" 등

예:
```
이름과 성: 유지현
조직 단위: 개발
조직: 유지현
도시: 서울
주/도: 서울
국가 코드: KR
```

## 중요 사항

⚠️ **이 정보는 키스토어에 저장되지만, 앱 기능에는 영향을 주지 않습니다**
- 앱이 정상 작동함
- 사용자에게 표시되지 않음
- Play Store 등록 시 참고 정보로만 사용

✅ **권장:** 회사/조직 정보를 입력하는 것이 좋습니다
